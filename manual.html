02 📍 장소 저장

자주 가는 배달지·건물을 저장해두면 가까워질 때 자동으로 메모를 음성 안내합니다.

방법 A — 현위치 저장 (추천)
  • 목적지 건물 앞에서
    📍 현위치 저장
    버튼 탭
  • 카카오 지도가 건물명 또는 도로명을 자동으로 입력란에 채웁니다
  • 메모 입력란에 참고 정보를 입력 (예: B1 / 경비실 앞)
  • 저장
    버튼 탭
방법 B — 직접 입력
  • 장소명 입력란에 건물명 직접 입력
  • 메모 입력란에 참고 정보 입력
  • 저장
    버튼 탭 — GPS 핀 없이 이름과 메모만 저장됩니다
GPS 핀 추가 (📍¹ / 📍²)

각 장소에는 최대 2개의 GPS 핀을 설정할 수 있습니다. 정문·후문처럼 출입구가 여러 개인 건물에 유용합니다.

  • 목록에서 장소 항목의 📌¹ 또는 📌² 버튼 탭
  • 현재 위치가 해당 핀으로 저장됩니다
  • 저장 완료 시 아이콘이 📍¹ 으로 바뀝니다
핀이 없으면 거리 정렬과 음성 안내가 작동하지 않습니다. 반드시 현장에서 핀 저장을 해두세요.
수정 · 삭제
  • 목록 항목 오른쪽 "수정" 버튼 → 입력란에 내용 불러옴 → 수정 후 저장
  • "삭제" 버튼 또는 항목 3초 꾹 누르기 → 확인 후 삭제

03 🃏 히어로 카드 · 음성 안내

화면 상단의 녹색 테두리 카드가 히어로 카드입니다. 가장 가까운 장소 정보를 대형 글씨로 표시합니다.

롯데월드타워
B1
0.02km 🔊 음성 안내
음성 안내 조건 (모두 충족 시 자동 발화)
  • 장소까지 거리 30m 이내 진입
  • 이동 속도 15km/h 이하
  • GPS 정확도 100m 이내
  • 해당 장소 첫 진입 시 1회 안내 (300m 이상 이격 후 재진입 시 다시 안내)
안전 모드 (기본 상태)

GPS 핀이 없거나 30m 이상 떨어져 있으면 히어로 카드는 안전 운행 메시지를 스크롤로 표시합니다. 카드를 탭하면 안전 모드 ↔ 활성 모드를 수동으로 전환할 수 있습니다.

05 📦 배달 카운터

화면 중앙의 큰 숫자가 오늘의 총 배달 건수입니다.

조작 방법
  • + 버튼 탭 → 건수 +1 (진동 + 비프음)
  • 층수 버튼 탭 후 해제 → 건수 자동 +1 (층수 안내도 함께)
  • − 버튼 탭 → 건수 −1 (잘못 누른 경우)
  • 숫자 탭 → Total Count ↔ 시간당 건수 (건/h) 모드 전환
5건마다 달성 알림이 울립니다 (미션 모드에서는 1건마다). 시간당 건수 모드에서는 숫자가 주황색으로 표시됩니다.

07 💰 시급 계산

STOP을 탭하면 수입 입력 팝업이 자동으로 나타납니다.

수입 입력 팝업
  • 오늘 정산 수입을 입력 (숫자만)
  • 시급 계산 탭 → 근무시간 기반 시급 즉시 표시
  • 저장 탭 → 기록 저장 (당일 여러 번 저장 시 누산)
  • 닫기 탭 → 저장 없이 종료

08 🚨 SOS · 긴급연락처

홈 화면 하단의 [긴급] 버튼을 탭하면 긴급연락처 페이지가 열립니다. 페이지 상단의 SOS 버튼은 응급 신고 보조 도구이며, 하단에서 자주 사용하는 연락처를 카테고리별로 관리할 수 있습니다.

SOS 버튼 사용법
  • 2초간 길게 누름 → 진행 게이지가 채워짐
  • 완료 시 진동 발생 + 현재 GPS 좌표가 클립보드에 복사
  • 음성으로 "위치를 클립보드에 복사했습니다" 안내
  • 확인 모달이 열림 → [다이얼 열기] 탭하면 119 다이얼 화면 호출
  • 다이얼 화면에서 통화 버튼을 직접 눌러 신고
  • 신고 시 복사된 좌표를 구두로 전달하거나 메시지로 붙여넣어 사용
⚠️ 자동 통화 없음: 본 앱은 임의로 119에 전화를 걸지 않습니다. 다이얼 화면 호출만 수행하며, 실제 통화 연결은 이용자가 직접 통화 버튼을 눌러야 합니다. 응급상황에서는 본 앱에 의존하지 말고 119에 직접 연락하세요.
긴급연락처 카테고리
  • 공공 기본 (수정 불가): 119 화재·구급 / 112 경찰 / 1339 응급의료상담
  • 플랫폼: 배달 플랫폼 고객센터
  • 보험: 보험사 사고접수 번호
  • 소속: 대리점·지사 연락처
  • 기타: 가족·지인 등 자유 등록
연락처 추가 · 수정 · 삭제
  • 추가: 카테고리 헤더의 [+] 버튼 탭 → 이름·번호 입력
  • 전화 걸기: 연락처 카드 탭 → 다이얼 화면 호출
  • 수정: 카드 우측 [편집] 아이콘 탭
  • 삭제: 편집 모드에서 [삭제] 탭
사용자 등록 연락처는 PIN 기반 AES-256 암호화로 기기 내에 저장됩니다. 운영자 서버에는 전송되지 않습니다.
09 🔔 알림 시스템

앱은 아래 상황에서 자동으로 진동·음성·팝업 알림을 발생시킵니다.

근무 알림
매 1시간 경과 근무 시간 + 건당 시간
2시간 연속 근무 5분 휴식 권장
5건 달성마다 건수 마일스톤
알림은 히어로 카드에도 스크롤 텍스트로 표시됩니다. 기기 알림 권한이 허용된 경우 시스템 알림도 함께 발생합니다.
10 💾 백업 · 복원

장소 데이터와 근무 기록을 암호화 파일로 내보낼 수 있습니다.

백업 (내보내기)
  • "기록" 버튼 탭 → 기록 화면 이동
  • "백업" 버튼 탭
  • hud_backup_YYYYMMDD.enc 파일이 다운로드됩니다
  • 파일을 클라우드 드라이브 등 안전한 곳에 보관하세요
복원 (불러오기)
  • "복원" 버튼 탭 → 파일 선택
  • 백업 생성 시와 동일한 PIN으로 잠금해제된 상태여야 합니다
  • 복원 시 현재 데이터가 모두 덮어씌워집니다 — 확인 팝업 꼭 읽기
백업 파일은 PIN으로 AES 암호화되어 있습니다. PIN이 다르면 복원이 불가능합니다. 기기 교체 시 반드시 백업 먼저 받으세요.
11 🔒 PIN · 보안
자동 잠금

백그라운드로 전환 후 일정 시간이 지나면 자동으로 잠금됩니다. 근무 중(타이머 실행 중)에는 자동 잠금이 작동하지 않습니다.

PIN 초기화

PIN을 잊어버린 경우 잠금 화면 하단 "PIN 초기화"를 탭하세요.

PIN 초기화 시 암호화된 장소 데이터 전체가 삭제됩니다. 백업이 없으면 복구 불가능합니다. 근무 기록은 선택적으로 유지할 수 있습니다.
데이터 보안 방식
암호화 AES-256 (CryptoJS)
키 파생 PBKDF2 (SHA-256, 10,000 rounds)
PIN 저장 SHA-256 해시 (원문 미저장)
서버 전송 없음 — 기기 내 전용
12 ❓ 자주 묻는 질문
음성 안내가 안 나와요
  • 기기 음량이 0인지 확인
  • GPS 핀이 저장되어 있는지 확인 (📌¹ → 📍¹)
  • 이동 속도가 15km/h를 초과하면 안내되지 않습니다
  • 같은 장소에서 이미 안내된 경우 → 300m 이상 이동 후 재진입 필요
GPS 신호가 약하다고 나와요
  • 실내·지하·터널에서는 GPS 수신이 어렵습니다
  • 경고 표시는 3회 연속 실패 시 나타납니다 — 야외로 이동하면 자동 해제
  • 브라우저 위치 권한이 "허용"으로 설정되어 있는지 확인하세요
데이터가 사라졌어요
  • PIN을 바꾸거나 초기화한 경우 장소 데이터가 삭제될 수 있습니다
  • 백업 파일이 있다면 복원 기능을 이용하세요
카카오 지도 주소 검색이 안 돼요
  • 네트워크 연결을 확인하세요
  • 카카오 API는 등록된 도메인에서만 작동합니다 (Netlify 배포 후 정상)
  • 로컬 파일(file://)로 열면 카카오 기능이 비활성화됩니다
저장 공간이 부족하다는 경고가 떠요

기록 화면에서 오래된 근무 기록을 백업 후 삭제하세요. 기기 브라우저 저장소 용량 제한에 도달한 상태입니다.
